welcome

In our templates you can easily make many changes. We prepared two version of this template: dark and bright.
We shall explain below how you can make changes in the:

- logo
- header text
- background
- about us text.
- testimonial text .
- social icons (we give you 21 different social icon to choose) .
- gallery.
- PHP for contact form and subscrive.

logo

To add your own logo first you should add your .png file to  images folder 
 
to fit your own logo open css file and modify some parameters:
 
.navbar-brand>img {
  1. display: block;
  2. margin-top: 14px;
  3. margin-left: 45px;
  4. padding-right: 50px;
}

background

.jumbotron {
